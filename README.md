# TaskMitra | Premium Service Marketplace for India

TaskMitra is a high-performance, mobile-first service marketplace tailored for the Indian market. It integrates the best features of TaskRabbit (task posting), UrbanCompany (trusted home services), JustDial (business discovery), and Swiggy Genie (hyper-local logistics) into a single, cohesive platform.

## 🚀 Key Features

- **Massive Service Directory**: Over 30+ categories including Furniture Assembly, Home Cleaning, Plumbing, Tutors, and Legal services.
- **Professional Dashboards**:
  - **User Dashboard**: Track active tasks, messages, and payment history.
  - **Pro Dashboard**: A live board for workers to find nearby jobs, manage schedules, and track earnings.
- **Worker Registration**: A streamlined 3-step onboarding flow for professionals with identity verification support.
- **V2 Corporate Aesthetic**: A crisp, trustworthy design system using a Deep Slate and Forest Green palette.
- **Swiggy Genie Inspired**: Dedicated flow for point-to-point delivery and quick errands.

## 🛠️ Tech Stack

- **React + Vite** (High-performance frontend)
- **Tailwind CSS V4** (Native utility styling)
- **Lucide React** (Consistent iconography)
- **React Router** (SPA navigation)

## 📱 Mobile Architecture

Designed with a mobile-first paradigm to ensure accessibility across diverse devices in India, featuring responsive grids and touch-optimized components.

## 📦 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/[REPO_OWNER]/task-mitra.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
